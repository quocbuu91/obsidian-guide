Bước 1 · Khởi đầu
Tải & Cài đặt Obsidian

Obsidian là công cụ ghi chú cục bộ (local-first) — dữ liệu lưu trên máy bạn, không phụ thuộc đám mây. Hoàn toàn miễn phí cho cá nhân.

  • ✓
    Truy cập obsidian.md → nhấn Download → chọn phiên bản phù hợp (Windows / macOS / Linux)
  • ✓
    Cài đặt ứng dụng và mở lần đầu tiên
  • ✓
    Chọn Create new vault khi được hỏi
  • ✓
    Đặt tên vault: startup-[tên-dự-án] và chọn thư mục lưu trữ
💡
Vault là gì? Vault = kho tài liệu. Mỗi vault là một thư mục trên máy tính. Bạn có thể có nhiều vault cho nhiều dự án khác nhau.
⚠️
Lưu ý: Đặt vault trong Google Drive / OneDrive nếu muốn tự động sao lưu lên đám mây — không cần phải trả phí Obsidian Sync.

Bước 2 · Nền tảng
Hiểu giao diện & Markdown cơ bản

Obsidian dùng ngôn ngữ Markdown — cực kỳ đơn giản, bạn sẽ thành thạo trong 10 phút.

📝
Editor (giữa)
Nơi bạn viết ghi chú. Nhấn Ctrl+E để chuyển giữa chế độ Edit và Preview.
🗂️
File Explorer (trái)
Cây thư mục dự án. Kéo thả file để sắp xếp, click phải để tạo folder mới.
🔗
Graph View (Ctrl+G)
Xem bản đồ liên kết giữa các ghi chú — điểm độc đáo nhất của Obsidian.
🔍
Quick Switch (Ctrl+O)
Mở nhanh bất kỳ file nào bằng tên — cực kỳ hữu ích khi vault lớn.

Markdown cần nhớ ngay:

Tiêu đề

→

**in đậm**
→

→

→

→

- [ ] Chưa xong
→

Liên kết — Quan trọng nhất ⭐

🎯
Thực hành ngay: Tạo file mới (Ctrl+N), thử gõ các cú pháp trên, nhấn Ctrl+E để xem kết quả render.

Bonus · Món quà cho bạn
Tạo toàn bộ cấu trúc trên trong 5 giây — không cần cài gì
1
Mở Obsidian → nhấn Ctrl + Shift + I (macOS: Cmd + Option + I) → cửa sổ Developer Tools hiện ra
2
Chọn tab Console ở thanh trên cùng
3
Nếu thấy cảnh báo màu vàng/đỏ — không lo, đây là cảnh báo bảo mật bình thường. Gõ allow pasting rồi nhấn Enter
4
Nhấn nút [Copy] ở góc phải trên của khung code bên dưới → Paste vào Console → nhấn Enter
⚠️ Phải dùng nút Copy — không bôi đen thủ công, sẽ bị lỗi!
✓
Console hiện chữ undefined — đó là bình thường, không phải lỗi! Nhìn góc phải trên Obsidian thấy thông báo ✅ và mở File Explorer kiểm tra thư mục đã được tạo.
📋 Copy toàn bộ đoạn này
// Code tạo toàn bộ cấu trúc vault startup // Bao gồm: 10 folders + 14 files có nội dung sẵn (async () => { // Bước 1: Tạo 10 thư mục const folders = [ "00-MOC", "01-Ideas", "02-Research", "02-Research/User-Interviews", "03-Product", "04-Business", "05-Team", "05-Team/Meeting-Notes", "06-Pitch", "07-Resources" ]; // Bước 2: Tạo 14 files với nội dung mẫu const files = { "00-MOC/🏠 Home.md": ← trang chủ vault "01-Ideas/Problem Statement.md": ← vấn đề cần giải quyết "01-Ideas/Solution Ideas.md": ← ý tưởng giải pháp "02-Research/Market Analysis.md": ← phân tích thị trường "02-Research/Competitor Map.md": ← bản đồ đối thủ "03-Product/MVP Scope.md": ← phạm vi MVP "03-Product/User Stories.md": ← user stories "03-Product/Roadmap.md": ← lộ trình phát triển "04-Business/BMC.md": ← business model canvas "04-Business/Financial Plan.md": ← kế hoạch tài chính "05-Team/Team Members.md": ← thành viên team "06-Pitch/Pitch Deck Outline.md": ← outline pitch deck "06-Pitch/QA Preparation.md": ← chuẩn bị Q&A "07-Resources/Useful Links.md": ← link hữu ích }; // Kết quả: thông báo ✅ ở góc phải trên })(); // ⚠️ KHÔNG copy từ màn hình này // → Nhấn nút [Copy] ở góc phải trên ↗️
⚡
Chỉ cần chạy 1 lần. Code tự bỏ qua folder/file đã tồn tại — không lo bị ghi đè nếu lỡ chạy lại.

Về cảnh báo "Don't paste code...": Đây là Chrome bảo vệ bạn khỏi code lạ từ internet. Đoạn code trên chỉ dùng API chính thức của Obsidian (app.vault.createFolder và app.vault.create) để tạo thư mục và file — không truy cập mạng, không gửi dữ liệu đi đâu.

Bước 5 · Templates
Tạo Templates cho mọi loại tài liệu

Templates giúp mỗi lần tạo tài liệu mới đều đúng format, tiết kiệm thời gian và đồng nhất cho cả team.

Bật Core Plugin Templates:

  • ✓
    Vào Settings → Core plugins → Templates → bật ON
  • ✓
    Tạo folder _Templates/ trong vault
  • ✓
    Trong Settings → Templates → chỉ đường dẫn đến folder _Templates

Điều hay của template: các biến tự điền

Bạn viết biến trong template, Obsidian tự thay bằng giá trị thật khi tạo note:

{{date}}
→

{{time}}
→
14:30 (giờ hiện tại)
{{title}}
→
Họp sprint 1 (tên file bạn đặt)

Bước 6 · Plugins
Cài Community Plugins thiết yếu

Obsidian có marketplace plugin mạnh mẽ. Đây là 5 plugin không thể thiếu cho startup — tất cả đều miễn phí.

📅 Dataview Quan trọng nhất
Query dữ liệu từ ghi chú như SQL. VD: liệt kê tất cả Action Items chưa xong trong mọi cuộc họp.
📋 Kanban
Tạo bảng Kanban (To Do / Doing / Done) ngay trong Obsidian — quản lý sprint của team.
📆 Calendar + Daily Notes
Ghi chú hằng ngày (daily log), liên kết với lịch — theo dõi tiến độ startup từng ngày.
🎨 Excalidraw
Vẽ sơ đồ, flowchart, wireframe ngay trong Obsidian. Cực hữu ích khi brainstorm cùng team.
🤖 Copilot (AI)
Tích hợp AI vào Obsidian — tóm tắt, phân tích ghi chú, brainstorm ý tưởng mới.
🔧
Cách cài: Settings → Community plugins → Turn off Safe mode → Browse → tìm tên plugin → Install → Enable.

Dataview làm được gì? Bạn viết câu lệnh → nhận kết quả sống:

Ví dụ: gom tất cả việc chưa làm (action items) rải rác trong mọi cuộc họp về một chỗ — tự cập nhật mỗi khi bạn tick xong.

⌨️ Bạn viết trong note
```dataview TASK WHERE !completed AND contains(tags, "#meeting") SORT file.mtime DESC ```
⬇️ Obsidian tự render thành ⬇️
✅ Kết quả hiển thị (tự cập nhật)
Gửi báo giá cho khách A Họp sprint 1
Sửa bug trang đăng nhập Họp kỹ thuật
Chuẩn bị slide pitch Họp với mentor
💡
Bạn không cần copy task thủ công — Dataview tự quét toàn vault. Tick xong ở note gốc thì nó tự biến mất khỏi danh sách này.

Bước 7 · Workflow
Thiết lập quy trình làm việc cho cả Team

Obsidian cá nhân thì tốt, nhưng dùng chung với team mới thực sự mạnh. Đây là workflow đề xuất cho startup 2–5 người.

Quy trình hằng ngày (Daily Workflow):

Sáng
Mở Daily Note (Ctrl+Shift+D) → ghi 3 mục tiêu hôm nay → link đến task liên quan
Trưa
Cập nhật Kanban board → tick task xong → ghi lại blockers
Tối
Quick retrospective 5 phút trong Daily Note → "Học được gì hôm nay?" → capture ý tưởng mới vào Inbox

Chia sẻ Vault với Team — 3 cách:

☁️ Google Drive / OneDrive Dễ nhất · Miễn phí
Để vault trong shared folder. Phù hợp người mới. Lưu ý: tránh 2 người mở/sửa cùng một file một lúc.
🐙 GitHub (plugin Obsidian Git) Mạnh nhất · Miễn phí
Version control đầy đủ: xem lịch sử thay đổi, ai sửa gì, khôi phục bản cũ. Phù hợp team có người rành kỹ thuật.
🔄 Obsidian Sync (chính chủ) Trả phí
Đồng bộ tức thì giữa các thiết bị, có mã hoá. Mượt nhất nhưng tốn phí hàng tháng — cân nhắc khi team đã ổn định.
